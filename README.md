Yii 2 REST
============================
Методы веб-сервиса:
* пользователь системы. Пользователь имеет обязательные атрибуты: имя, фамилия, день рождения, аватар. Все остальные атрибуты дополняются по желанию

* проект. Обязательные атрибуты: наименование, дата создания, дата планируемого окончания проекта, описание, владелец проекта, статус проекта.

* задание. Задание создается в рамках проекта, и назначается определенному пользователю, имеет наименование, описание, исполнителя, статус, директивные сроки. Пользователь может переназначить задание другому пользователю, при этом обязательно наличие комментария и сохранение в истории пред исполнителя.

Веб-сервис должен позволять:

* авторизоваться пользователям (HTTP Basic || Oauth)

* просмотр списка и карточки своих проектов (созданные и назначенные как исполнитель)

* создать, редактировать, удалять проекты (удаление проектов только логическое)

* просмотр списка и карточки своих заданий

* создать, редактировать, удалять задания к проектам (удаление заданий только логической)

* редактировать свой профиль


CONFIGURATION
-------------

### Database

Edit the file `config/db.php` with real data, for example:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2basic',
    'username' => 'root',
    'password' => '1234',
    'charset' => 'utf8',
];
```

**NOTES:**
- Yii won't create the database for you, this has to be done manually before you can access it.
- Check and edit the other files in the `config/` directory to customize your application as required.
- Refer to the README in the `tests` directory for information specific to basic application tests.
